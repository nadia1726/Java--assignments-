 Java Assignment: Basic Calculator Program Objective
To build a basic arithmetic calculator using Java's `Scanner` class and arithmetic operators (`+`, `-`, `*`, `/`).

Problem Statement
Write a Java program that takes two numbers as input from the user and performs addition, subtraction, multiplication, and division.

Sample Input/Output

Enter first number: 10
Enter second number: 5
Addition: 15.0
Subtraction: 5.0
Multiplication: 50.0
Division: 2.0
