Topic: Control Statements in Java (If-Else)

Question: Write a Java program that takes a person's age as input from the user and checks whether they are eligible to vote or not.
Condition: A person is eligible to vote if their age is 18 or older. Otherwise, they are not eligible.
Requirements:
Use the Scanner class to take input.
Use an if-else statement to implement the logic.
Make sure to close the scanner object properly before the program ends.
Sample Input/Output 1:
Enter your age: 20
Eligible to vote.
