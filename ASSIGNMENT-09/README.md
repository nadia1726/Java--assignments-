Write a Java program that uses a switch statement to display a language based on the user's choice.
Requirements:
Ask the user to enter an option from 1 to 3.
Use a switch statement to determine the selected language.
Display the following:
If the user enters 1, print: Selected language is Bengali
If the user enters 2, print: Selected language is Hindi
If the user enters 3, print: Selected language is Urdu
For any other input, print: Selected language is English
Use the Scanner class to take input from the user.
Close the Scanner object after use.
